bước 1: vào mục stories mong muốn trên trình duyệt, nhấn chuột phải chọn "view source page"
***VÀO FOLDER MAIN*** (đây là cho chế độ facebook, chọn mấy cái khác cũng được, nhưng chế độ facebook khá phức tạp nên có tuto riêng)
bước 2: copy vào "source.txt"
bước 3: tải extension "https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc" trong trình duyệt
bước 4: vào facebook, bật extension lên và nhấn nút "copy"
bước 5: nhập cookies đó vào file "cookies.txt" trong folder "main" (cookies này sau một thời gian sẽ hết hạn, cần thay mới)
bước 6: ra folder "download" rồi nhấn "run.bat" và chọn nhập link thủ công hoặc tự động từ file "link.txt" trong folder getlink 
bước 7: nhấn vào folder "finish" để xem

NOTE: Bật server trong terminal để có thể tải video từ các thiết bị khác trong cùng mạng
